# Soontech Annals Archive

A curation of encoded technologies in Minecraft

## Naming convention

First two letters indicate category. Next two numbers indicate index (incrementing by 1 with each machine).

| Category              | Code |
|-----------------------|------|
| box-variable          | BV   |
| chest-halls           | CH   |
| data-storage          | DS   |
| decoders              | DC   |
| display-halls         | DH   |
| encoded-systems       | ES   |
| encoders              | EC   |
| everything-bagel      | MX   |
| external-bulk         | EB   |
| item-memory           | IM   |
| item-variable         | IV   |
| logic-and-computation | LC   |
| merging               | MG   |
| splitting             | SG   |
| transport             | TP   |
| wiring                | WI   |
